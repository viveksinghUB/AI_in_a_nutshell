Simple Linear Regression


Introduction

Linear Regression is made up of two words : Linear, meaning "resembling a line" and Regression, that is "a technique to determine the strength of the relationship between one dependent variable and a series of other changing variables (known as independent variables)" .
In short, it is a technique that identifies a linear relation between two variables.The two variables that we are talking about are independent and dependent variables.
Dependent variable is a variable (often denoted by y ) whose value depends on that of another.It is called dependent because it "depends" on the independent variable.

If the number of independent variable is more than one then the technique is called Multivariate Linear Regression.
Nothing to fear from big names ;) Multivariate=> Multi meaning many & variate meaning variable, thus Many Variable Linear Regression.

Why Linear Regression

Lets say you give an exam having 10 questions. The number of questions answered and total score you get is as below.
Note that the marks for each question are not equal

Total questions answered Total Marks obtained
1 10
2 12
5 27
9 30
10 50

Now, if I give you this table and the plot and ask you, What do you think I will get if I had answered 4 questions ? Approximately 20, correct. There you go, you just predicted the value of dependent variable using Linear regression :)
When the data points are placed in a way that a line/plane can pass through/near most of the data points, linear regression is used.

We will discuss the technique on how to find the best fit line to answer our question.



Algorithm

We need to find such a line, that passes through most of the data points and is close to other points as well, so that given a value of x (the independent variable) we can predict the value of y (the dependent variable) with minimum error.

Mathematically we form an error function (known as cost function)that depicts the error in predicted value and actual value. Lets suppose, we choose the line " y=2x" as our best fit line for the above example.

Total questions answered (x) Total Marks obtained (y) Total Marks predicted (y'=2x) Error(y'-y) Squared error
1 10 2 -8 64
2 12 4 -8 64
5 27 10 -17 289
9 38 18 -20 400
10 50 20 -30 900
Total: 1717


We define our cost function as the Mean squared error , that is nothing but Total/(2*number of data points).
We want to find out the equation of such a line that gives minimum mean squared error.
Note that y' is the predicted value of y corresponding to an independent variable x. In short, y'=mx+c.
If number of independent variables are multdimension, then it becomes y'=EquationOfPlane(x1,x2,x3...).
Thus, we need to find out such coefficients of independent variable (such as m and c in case of equation of a line, y=mx+c ), for which the value of cost function is minimum. This is easily achievable by Gradient Descent.. For now, think of it as a black box where we feed the cost function and the derivative of cost function, and it spits out the coefficients of the function which gives the minimum error value for all data point.



Best Fit Line

We saw that using gradient descent, we get the coefficients of equation that gives minimum mean squared error. The line represented by coefficients is known as Best Fit Line. As in, the line that best fits the data.

Applications:

1) Gradient descent is generaly used to find the value of parameters which result in minimum error.

Source: http://wiki.fast.ai/index.php/Gradient_Descent

For example, if we want to find the line which best fits the data, we would form a cost function that measures the distance of point from the line as error. If the point lies on the line the error is zero, the more the distance of point from the line, the more is the error.
For a given set of points, we form an error function, and implement a gradient descent to find such parameters of the line which results in minumum overall error. I have discussed more on this in my tutorial on Linear regression.

2)By the sign in our update rule to w:=w+α(dEdw) , gradient descent can be changed to gradient ascent to find the maxima of a function.
